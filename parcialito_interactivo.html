üîÑ M√©todo de Gauss-Seidel

Problema: Resolver el sistema Ax = b usando Gauss-Seidel
A = [[4,-1,0], [2,7,1], [0,-3,5]]
b = [15, 20, 10]
Condiciones: x‚ÇÄ = [0,0,0], tolerancia = 10‚Åª‚Å∂, m√°x 100 iteraciones
üéØ Estrategia de implementaci√≥n:
Vamos a usar el enfoque matricial moderno para Gauss-Seidel. Primero descomponemos A = L + D + U (triangular inferior + diagonal + triangular superior). Luego reformulamos el problema como (D+L)x = -Ux + b, que nos da la iteraci√≥n x^(k+1) = M_GS ¬∑ x^(k) + c_GS. Este enfoque es m√°s eficiente que el elemento-a-elemento porque cada iteraci√≥n es solo una multiplicaci√≥n matriz-vector. Adem√°s, la matriz M_GS nos permite analizar la convergencia: si su radio espectral es menor a 1, el m√©todo converge. Esperamos que converja m√°s r√°pido que Jacobi porque usa valores actualizados inmediatamente.
Nivel 1: Esquema General
Enfoque matricial moderno:
1. Descomponer A = L + D + U (inferior + diagonal + superior)
2. Reformular: (D+L)x = -Ux + b
3. Construir matriz de iteraci√≥n: M_GS = -(D+L)‚Åª¬πU
4. Vector constante: c_GS = (D+L)‚Åª¬πb
5. Iterar: x^(k+1) = M_GS ¬∑ x^(k) + c_GS
6. Verificar convergencia con ||x^(k+1) - x^(k)|| < tol
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ D = np.diag(np.diag(A)) - extraer diagonal
‚Ä¢ L = np.tril(A, -1) - triangular inferior sin diagonal
‚Ä¢ U = np.triu(A, 1) - triangular superior sin diagonal
‚Ä¢ Verificar: A = L + D + U debe ser True
‚Ä¢ M_GS = -np.linalg.inv(D + L) @ U
‚Ä¢ Una multiplicaci√≥n matriz-vector por iteraci√≥n
Nivel 3: C√≥digo Completo
def gauss_seidel_matricial(A, b, x0, tol, max_iter): # Descomposici√≥n A = L + D + U D = np.diag(np.diag(A)) L = np.tril(A, -1) # k=-1 excluye diagonal U = np.triu(A, 1) # k=1 excluye diagonal # Verificar descomposici√≥n assert np.allclose(A, L + D + U) # Matriz de iteraci√≥n y vector constante M_gs = -np.linalg.inv(D + L) @ U c_gs = np.linalg.inv(D + L) @ b # Iteraci√≥n x = x0.copy() for k in range(max_iter): x_anterior = x.copy() x = M_gs @ x + c_gs if np.linalg.norm(x - x_anterior, np.inf) < tol: return x, k+1, np.linalg.eigvals(M_gs) return x, max_iter, np.linalg.eigvals(M_gs) # Radio espectral para verificar convergencia A = np.array([[4,-1,0],[2,7,1],[0,-3,5]], dtype=float) b = np.array([15,20,10], dtype=float) sol, iter, eigenvals = gauss_seidel_matricial(A, b, [0,0,0], 1e-6, 100) print(f"Radio espectral: {max(abs(eigenvals)):.6f}")

‚Ä¢ Crear vector t = np.arange(0, 2.1, 0.1)
‚Ä¢ Inicializar y = np.zeros(len(t)), y[0] = 0

üìè Estimaci√≥n de Norma-2

Matriz A = [[3,1,-1], [1,4,2], [-1,2,5]]
Objetivo: Estimar ||A||‚ÇÇ con m√©todo iterativo
Algoritmo: s‚ÇÅ = 0, s_{k+1} = max{s_k, ||Ax_k||‚ÇÇ/||x_k||‚ÇÇ}
Iteraciones: 50
üéØ Estrategia de implementaci√≥n:
Queremos estimar cu√°nto puede "estirar" esta matriz a un vector. La norma-2 nos dice el m√°ximo factor de estiramiento posible. Como no podemos probar todos los vectores infinitos, vamos a hacer un truco: generar vectores aleatorios, aplicarles la matriz, y ver cu√°nto se estiraron. Vamos a mantener una "marca personal" (el m√°ximo estiramiento visto hasta ahora) que iremos actualizando. Con suficientes intentos aleatorios, deber√≠amos acercarnos bastante a la norma real. Al final compararemos nuestro resultado con el valor exacto que calcula NumPy.
Nivel 1: Esquema General
1. Generar vectores aleatorios x_k unitarios
2. Calcular ||A¬∑x_k||‚ÇÇ
3. s_k = max{s_{k-1}, ||A¬∑x_k||‚ÇÇ}

5. Comparar con np.linalg.norm(A, 2)
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ x = np.random.randn(n), luego x = x/np.linalg.norm(x)
‚Ä¢ La sucesi√≥n s_k debe ser mon√≥tona creciente
‚Ä¢ Graficar convergencia vs valor exacto
‚Ä¢ np.random.seed() para reproducibilidad
Nivel 3: C√≥digo Completo
def estimar_norma(A, n_iter): s = np.zeros(n_iter) for k in range(1, n_iter): x = np.random.randn(A.shape[0]) x = x / np.linalg.norm(x) s[k] = max(s[k-1], np.linalg.norm(A @ x)) return s A = np.array([[3,1,-1],[1,4,2],[-1,2,5]]) s = estimar_norma(A, 50) plt.plot(s) plt.axhline(np.linalg.norm(A,2), color='r', linestyle='--')

üî¢ Diferencias Finitas

EDP: -u'' + 2u = x¬≤ en [0,1]
Condiciones frontera: u(0) = 1, u(1) = 3
Discretizaci√≥n: N = 10 puntos internos
Objetivo: Encontrar u(0.5)
üéØ Estrategia de implementaci√≥n:
Tenemos una ecuaci√≥n diferencial de segundo orden en el espacio, no en el tiempo. Vamos a dividir el intervalo [0,1] en pedacitos y aproximar la segunda derivada usando diferencias finitas. Como conocemos los valores en los extremos (u(0)=1, u(1)=3), solo necesitamos calcular los valores en los puntos internos. Esto nos va a dar un sistema de ecuaciones lineales: una ecuaci√≥n por cada punto interno. La matriz ser√° tridiagonal (cada punto depende solo de sus vecinos). Una vez que resolvamos el sistema, tendremos la soluci√≥n aproximada en todos los puntos.
Nivel 1: Esquema General
1. u''·µ¢ ‚âà (u·µ¢‚Çä‚ÇÅ - 2u·µ¢ + u·µ¢‚Çã‚ÇÅ)/h¬≤
2. Ecuaci√≥n: (-2/h¬≤ + 2)u·µ¢ + (1/h¬≤)(u·µ¢‚Çã‚ÇÅ + u·µ¢‚Çä‚ÇÅ) = x·µ¢¬≤
3. Armar matriz tridiagonal A
4. Incorporar condiciones de frontera en vector b
5. Resolver Au = b
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ h = 1/(N+1), x = np.linspace(0,1,N+2)
‚Ä¢ Diagonal principal: -2/h¬≤ + Œ±
‚Ä¢ Super/sub diagonales: 1/h¬≤
‚Ä¢ b[0] -= u0/h¬≤, b[-1] -= u1/h¬≤
‚Ä¢ Concatenar soluci√≥n: [u0, u_interno, u1]
Nivel 3: C√≥digo Completo
def diferencias_finitas(N, alpha, u0, u1): h = 1.0 / (N + 1) x = np.linspace(0, 1, N + 2) A = np.zeros((N, N)) for i in range(N): A[i,i] = -2/h**2 + alpha if i > 0: A[i,i-1] = 1/h**2 if i < N-1: A[i,i+1] = 1/h**2 b = x[1:-1]**2 # f(x) = x¬≤ b[0] -= u0/h**2 b[-1] -= u1/h**2 u_int = np.linalg.solve(A, b) return x, np.concatenate([[u0], u_int, [u1]])

‚¨áÔ∏è Eliminaci√≥n Gaussiana

Sistema: Ax = b
A = [[2,1,-1], [4,5,3], [2,1,4]]
b = [8, 9, 11]
M√©todo: Eliminaci√≥n gaussiana SIN pivoteo
Objetivo: Mostrar matriz escalonada y soluci√≥n
üéØ Estrategia de implementaci√≥n:
Vamos a resolver este sistema usando el m√©todo cl√°sico que aprendiste en √°lgebra lineal, pero implement√°ndolo paso a paso. Primero vamos a hacer "ceros" debajo de la diagonal principal: usaremos la primera fila para eliminar elementos de las filas siguientes, luego la segunda fila para eliminar de las que quedan, y as√≠. Una vez que tengamos la matriz triangular superior, vamos a resolver "hacia atr√°s": empezamos por la √∫ltima ecuaci√≥n (que solo tiene una inc√≥gnita), la resolvemos, y vamos subiendo sustituyendo los valores ya encontrados. Es como desarmar un problema complejo en pasos m√°s simples.
Nivel 1: Esquema General
Dos etapas:
1. Eliminaci√≥n hacia adelante: crear ceros bajo la diagonal
2. Sustituci√≥n hacia atr√°s: resolver desde la √∫ltima ecuaci√≥n
‚Ä¢ Factor = A[i,k]/A[k,k]
‚Ä¢ Fila[i] = Fila[i] - factor √ó Fila[k]
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ Trabajar con copias: A.copy().astype(float)
‚Ä¢ For k in range(n-1): para cada pivote
‚Ä¢ For i in range(k+1,n): filas debajo del pivote
‚Ä¢ Actualizar tanto A[i,j] como b[i]
‚Ä¢ Sustituci√≥n: x[i] = (b[i] - suma)/A[i,i]
Nivel 3: C√≥digo Completo
def eliminacion_gaussiana(A, b): n = len(A) A = A.copy().astype(float) b = b.copy().astype(float) # Eliminaci√≥n hacia adelante for k in range(n-1): for i in range(k+1, n): factor = A[i,k] / A[k,k] for j in range(k, n): A[i,j] -= factor * A[k,j] b[i] -= factor * b[k] # Sustituci√≥n hacia atr√°s x = np.zeros(n) for i in range(n-1, -1, -1): x[i] = b[i] for j in range(i+1, n): x[i] -= A[i,j] * x[j] x[i] /= A[i,i] return A, x

üîÑ M√©todo de Jacobi

Sistema: Ax = b
A = [[10,-1,2], [-1,11,-1], [2,-1,10]]
b = [6, 25, -11]
Condiciones: x‚ÇÄ=[0,0,0], tol=10‚Åª‚Å∏, m√°x 1000 iter
Objetivo: Verificar convergencia y comparar con Gauss-Seidel
üéØ Estrategia de implementaci√≥n:
Vamos a implementar Jacobi usando el enfoque matricial, que es m√°s elegante que el elemento-a-elemento. Usamos la misma descomposici√≥n A = L + D + U, pero ahora agrupamos diferente: Dx = -(L+U)x + b, lo que nos da x^(k+1) = M_J ¬∑ x^(k) + c_J. La diferencia clave con Gauss-Seidel es que Jacobi usa TODOS los valores de la iteraci√≥n anterior, lo que lo hace paralelizable pero generalmente m√°s lento. Vamos a comparar su radio espectral con Gauss-Seidel para ver por qu√© uno converge m√°s r√°pido que el otro. El m√©todo es ideal cuando queremos implementaci√≥n paralela.
Nivel 1: Esquema General
Diferencia clave con Gauss-Seidel:
‚Ä¢ Jacobi: usar TODOS los valores de iteraci√≥n anterior x^k
‚Ä¢ No actualizar componentes durante la misma iteraci√≥n
‚Ä¢ Calcular completamente x^{k+1} antes de siguiente iteraci√≥n
‚Ä¢ Generalmente converge m√°s lento
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ Misma descomposici√≥n que Gauss-Seidel: A = L + D + U
‚Ä¢ M_J = -np.linalg.inv(D) @ (L + U)
‚Ä¢ c_J = np.linalg.inv(D) @ b
‚Ä¢ Comparar radio espectral: œÅ(M_J) vs œÅ(M_GS)
‚Ä¢ Paralelizable: cada componente se calcula independientemente
‚Ä¢ Generalmente œÅ(M_J) > œÅ(M_GS) ‚Üí converge m√°s lento
Nivel 3: C√≥digo Completo
def jacobi(A, b, x0, tol, max_iter): n = len(A) x = x0.copy() for k in range(max_iter): x_nuevo = np.zeros(n) for i in range(n): suma = 0 for j in range(n): if i != j: suma += A[i,j] * x[j] # valores anteriores x_nuevo[i] = (b[i] - suma) / A[i,i] if np.linalg.norm(x_nuevo - x) < tol: return x_nuevo, k+1, True x = x_nuevo.copy() return x, max_iter, False A = np.array([[10,-1,2],[-1,11,-1],[2,-1,10]], dtype=float) b = np.array([6,25,-11], dtype=float) sol, iter, convergio = jacobi(A, b, [0,0,0], 1e-8, 1000)

üîÑ Matriz Inversa

Matriz A = [[4,7], [2,6]]
M√©todo: Eliminaci√≥n Gauss-Jordan
Objetivo: Calcular A‚Åª¬π y verificar A¬∑A‚Åª¬π = I
Verificaci√≥n: Comparar con np.linalg.inv(A)
üéØ Estrategia de implementaci√≥n:
Vamos a calcular la inversa usando una extensi√≥n elegante de la eliminaci√≥n gaussiana. La idea es resolver m√∫ltiples sistemas a la vez: en lugar de resolver Ax=b para un solo vector b, vamos a resolver para todos los vectores de la matriz identidad simult√°neamente. Construimos una matriz "aumentada" [A|I] y aplicamos operaciones de fila hasta convertirla en [I|A‚Åª¬π]. Es como si estuvi√©ramos preguntando: "¬øqu√© matriz, multiplicada por A, da la identidad?". Al final vamos a verificar que nuestro resultado es correcto multiplicando A por nuestra inversa calculada.
Nivel 1: Esquema General
Proceso Gauss-Jordan: [A|I] ‚Üí [I|A‚Åª¬π]
1. Formar matriz aumentada [A|I]
2. Para cada fila i: normalizar dividiendo por A[i,i]
3. Eliminar columna i en todas las otras filas
4. Resultado: lado derecho es A‚Åª¬π
5. Verificar: A @ A_inv ‚âà I
Nivel 2: Tips de Implementaci√≥n
‚Ä¢ A_aug = np.hstack([A.copy(), np.eye(n)])
‚Ä¢ Para fila i: A_aug[i] = A_aug[i] / A_aug[i,i]
‚Ä¢ Para otras filas j‚â†i: A_aug[j] -= A_aug[j,i] * A_aug[i]
‚Ä¢ Verificar det(A) ‚â† 0 antes de empezar
‚Ä¢ Retornar A_aug[:, n:] (lado derecho)
Nivel 3: C√≥digo Completo
def matriz_inversa(A): n = len(A) # Verificar que es invertible if abs(np.linalg.det(A)) < 1e-10: return None # Matriz aumentada [A|I] A_aug = np.hstack([A.copy().astype(float), np.eye(n)]) # Gauss-Jordan for i in range(n): # Normalizar fila i A_aug[i] = A_aug[i] / A_aug[i,i] # Eliminar columna i en otras filas for j in range(n): if i != j: A_aug[j] = A_aug[j] - A_aug[j,i] * A_aug[i] return A_aug[:, n:] A = np.array([[4,7],[2,6]]) A_inv = matriz_inversa(A) print("A‚Åª¬π =", A_inv) print("Verificaci√≥n:", A @ A_inv)
